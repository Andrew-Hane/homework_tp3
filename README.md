# Домашнее задание 3: Docker и Bash

Проект генерирует мок-данные по фермам и кормам, а затем строит по ним HTML-отчет, используя изолированные Docker-контейнеры.

## Инструкция по запуску:

1. Соберите образ генератора:
   `./run.sh build_generator`
2. Сгенерируйте данные (файл появится в папке data/):
   `./run.sh run_generator`
3. Соберите образ аналитика:
   `./run.sh build_reporter`
4. Создайте отчет (файл report.html появится в папке data/):
   `./run.sh run_reporter`

После выполнения всех шагов файл `data/report.html` будет готов к просмотру в браузере.
