# Домашнее задание 3: Docker и Bash

Проект генерирует данные по фермам и кормам, а затем строит по ним HTML-отчет, используя изолированные Docker-контейнеры.

## Инструкция по запуску в GitHub Codespaces:

1. Соберите образы и сгенерируйте данные:
   `./run.sh build_generator`
   `./run.sh run_generator`
2. Соберите образ аналитика и создайте отчет:
   `./run.sh build_reporter`
   `./run.sh run_reporter`
4. В нижней панели терминала Codespaces перейдите во вкладку **"Ports"** (Порты).
5. Найдите проброшенный порт **8080**. Нажмите на иконку глобуса ("Open in Browser").
6. В открывшейся вкладке нажмите на файл `report.html`, чтобы посмотреть статистику.
